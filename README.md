# Genshin Impact Clone

A high-fidelity clone of the Genshin Impact official website.

## Features
- **Immersive Hero Section**: Video background and particle effects.
- **Interactive Gameplay Section**: Swiper carousel for screenshots and video modal.
- **Explore Section**: Region-specific backgrounds (Mondstadt, Liyue, Inazuma, Sumeru, Fontaine, Natlan, Nod-Krai).
- **Login System**: LocalStorage-based authentication simulation.
- **Responsive Design**: Optimized for different screen sizes.

## Technologies
- HTML5
- CSS3 (Custom Properties & Animations)
- JavaScript (Vanilla)
- Three.js (Footer Effects)
- Swiper.js
